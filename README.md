# 🌐 WebBuilder AI

AI-powered website generator. Describe your website and get instant UI with live preview and downloadable code.

---

## ✨ Features

* Generate website using Gemini AI
* Live preview
* Open in new tab
* Download code
* Show generated code
* Built with React + Tailwind

---

## 🛠️ Tech Stack

* React + Vite
* Tailwind CSS
* Gemini AI API

---

## 🚀 Run Locally

```bash
npm install
npm run dev
```

---

## 🌍 Deploy on Netlify

Build command:

```bash
npm run build
```

Publish directory:

```bash
dist
```
